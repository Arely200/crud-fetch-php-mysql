# CRUD de Productos con Fetch API + PHP OOP + MySQL

## Descripción
Aplicación web dinámica que implementa operaciones CRUD (Crear, Leer, Buscar, Actualizar) para un catálogo de productos.

## Tecnologías utilizadas
- **Frontend:** HTML5, Bootstrap 5, JavaScript (Fetch API), SweetAlert2
- **Backend:** PHP 8 (Programación Orientada a Objetos)
- **Base de datos:** MySQL (PDO)

## Funcionalidades
- Guardar nuevos productos (validación: cantidad mínima 1)
- Buscar productos por código
- Modificar productos existentes (cantidad puede ser 0)
- Listar todos los productos
- Validaciones en cliente y servidor
- Alertas interactivas con SweetAlert2
